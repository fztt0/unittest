# unittest
Automated Testing Activity 01
TDD:
Como usuário gostaria de uma funcionalidade que valide se uma frase ou palavra é palíndromo.

Critérios de aceite:

Verdadeiro:
- “Rotator”
- “bob”
- “madam”
- “mAlAyAlam”
- “1”
- “Able was I, ere I saw Elba”
- “Madam I’m Adam”
- “Step on no pets.”
- “Top spot!”
- “02/02/2020”

Falso:
- “xyz”
- “elephant”
- “Country”
- “Top . post!”
- “Wonderful-fool”
- “Wild imagination!”

O componente deve receber os exemplos acima e retornar verdadeiro ou falso, conforme a regra.
